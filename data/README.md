# Data Files

This directory contains all data files used by SpeedFog.

## Setup

Most data files are extracted from FogRando and are **not tracked in git**.
Run the setup script after downloading FogRando from Nexusmods:

```bash
python tools/setup_dependencies.py /path/to/FogRando.zip
```

This extracts source files and regenerates derived files automatically.

## Source Files (from FogRando, gitignored)

These files are extracted from FogRando by `setup_dependencies.py`:

| File | Description |
|------|-------------|
| `fog.txt` | Zone definitions, fog gates, warps, key items (YAML) |
| `fogevents.txt` | EMEVD event templates |
| `foglocations2.txt` | Enemy areas and scaling tiers (YAML) |
| `er-common.emedf.json` | EMEVD instruction definitions |

## Generated Files (gitignored)

These files are generated by Python tools during setup:

| File | Generated by | Description |
|------|--------------|-------------|
| `clusters.json` | `tools/generate_clusters.py` | Zone clusters with entry/exit fogs |
| `fog_data.json` | `tools/extract_fog_data.py` | Fog gate metadata for C# writer |

To regenerate manually:
```bash
python tools/generate_clusters.py data/fog.txt data/clusters.json --metadata data/zone_metadata.toml
python tools/extract_fog_data.py data/fog.txt data/fog_data.json
```

## Configuration Files (tracked in git)

| File | Description |
|------|-------------|
| `zone_metadata.toml` | Zone weight defaults and per-zone overrides |

## Runtime Output

The DAG generator outputs files to `seeds/<seed>/`:
- `graph.json` - Generated DAG consumed by C# writer
- `spoiler.txt` - Human-readable path visualization
